SDK 更新日志

Altizure Sandbox 3D SDK 更新日志

0.2.24

发布时间: 2018年6月20日

新功能:

  • 新版的sandbox.camera.lookAt，可以调整camera使得marker在视野中心。增加了範例5.10。

改进:

  • 优化了PolyFenceLineMarker的贴图逻辑，避免贴图错误。

缺陷修复:

  • 修复了IE11对 Math.log2 未定义的报错。

0.2.20

发布时间: 2018年5月24日

新功能:

  • Tag 类 Marker 增加 isSprite 属性，区分是否总面向镜头。更新 范例 2.2*。
  • Tag 类增加 pivot -- 面向镜头时的旋转中心，详见文档。
  • AltizureProjectMarker 初始化选项 autoscale: {number | bool} 自动缩放CAD类模型。范例 2.1.2

  • 根据天际线的区域限高。范例 6.6

改进:

  • 保留主站裁剪效果。
  • 加入cdn自动发布流程。
  • 集中处理离线渲染。
  • 提高quality上限。
  • 添加瓦片bbox显示接口 showTileBBox。
  • CameraMatrix重构及用例测试。

缺陷修复:

  • 当有重叠出现，positionsAltitude 返回错误高度。
  • GeoSystem 对齐后， Marker::position 未更新。
  • CAD模型的材质显示错误。
  • Marterial array 导致的 marker::destruct 报错。
  • 销毁项目模型时释放裁剪Mask内存。

0.2.5

发布时间: 2018年4月9日

新功能:

  • 区域标注 Zone Marker。范例 2.11
  • 建筑对象化。 範例 6.4
  • 标注聚合。範例 6.5
  • 质量-效率平衡选项 sandbox.quality。
  • 高进度投影坐标系 GeoSystem。
  • CAD转换
  • 交互式可视域分析。範例5.9

改进:

  • threejs 更新至0.91.0
  • 提高渲染效率。

缺陷修复:

  • LightBeam color 接口，接受string类型。

0.1.62

发布时间: 2018年2月8日

新功能:

  • Shadow Map 和通视分析。范例 5.8

改进:

  • 提高渲染效率。

缺陷修复:

  • 修复触碰手势和鼠标点击事件无响应的bug。
