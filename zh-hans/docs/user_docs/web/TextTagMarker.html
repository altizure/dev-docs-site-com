    TextTagMarker(markerOptions)

    new TextTagMarker(markerOptions)

    Parameters:
    Name Type Description
    markerOptions object
    Properties
    Name Type Attributes Default Description
    textOptions.text string

    textContent
    textOptions.textStyle object <optional>

    text style

    Properties
    Name Type Attributes Default Description
    fillStyle object <optional>
    		 "black"
    font object <optional>
    		 "Arial"
    outlineWidth object <optional>
    		 2
    textOptions.textStyle object <optional>
    		 "black"
    Example

    construction 

     let textTagMarker = new altizure.TextTagMarker({
   // text string
   text: 'Altizure',
   // text style
   textStyle: {
     fillStyle: 'rgb(155, 155, 255)',
     font: 'normal 500 24px Arial',
     outlineWidth: 2,
     outlineStyle: 'rgb(0, 0, 0)'
   },
   // icon position
   position: {
     "lng": 113.939420,
     "lat": 22.5362742,
     "alt": 32.40720
   },
   // scene
   sandbox: sandbox,
   scale: 1 // icon size
 })

    Do depth test or not. If false, always in front.

    Overrides:
    • SpriteMarker#depthTest
    Example

    Change depth test status 

     spriteMarker.depthTest = true // do depth test
 spriteMarker.depthTest = !spriteMarker.depthTest // flip the status

    euler :object

    The orientation in Eular angle representation. { x: rotation angle around X axis in radian, y: radian, z: radian, order: rotation order, default 'XYZ'}. Use this instead of orientation if you are not familar with the math.

    Inherited From:
    • Marker#euler
    See:
    • Eular angle rotation
    Example

    Set orientation by Eular angles 

     marker.euler = {x: Math.PI/6, y: Math.PI/4, z: Math.PI/3} // rotate around X-axis for 30 degree; Y for 45 degree; Z for 60 degree, in that order.
 marker.euler = {y: Math.PI/2} // rotate around Y-axis for 90 degree.

    event handler

    Register interaction events

   console.log('click on marker ', marker, event)
 })

   console.log('mouse over marker ', marker, event)
 })
 marker.on('mouseenter', function (event) {
   console.log('mouse enter marker ', marker, event)
 })
 marker.on('mouseleave', function (event) {
   console.log('mouse leave marker ', marker, event)
 })
 marker.interactable = true // the marker must be interactable for events to take effects
